
# Project Title

A simple API which translates every string you send to groot!


## API Reference

#### Translate
```
  GET /str
```

| Parameter | Type     | Description                |
| :-------- | :------- | :------------------------- |
| `inputString` | `string` | **Required**. str to translate to groot. |
